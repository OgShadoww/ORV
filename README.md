# ORV Chapter Reteller
A local-AI powered retelling tool for Omniscient Readers Viewpoint

This proect automatically fetches chapters from ORV and generates a clean, simple, engaging retelling of each chapter using a local Ollama LLM model (Phi-3, Llama-3, Mistral, etc.).

Its designed for rereading or catching up from where you left off  without reading massive full chapters again.
